QuickTime 扩展

QuickTime 扩展都做些什么?

QuickTime 扩展允许你在一个网上或者电影文件中插入或者创建一个图像。

安装 QuickTime 扩展

视窗

第一，确保你有装QuickTime ，如果你没有可以从这里下载：

http://www.apple.com/quicktime/download/

为了让视频扩展可以使用，我们要将装入QuickTime得一些文件移动到扩展词典里。

  • 复制 C:\Program Files\QuickTime\QTSystem\QTJava.zip到扩展字典，然后重命名为QTJava.jar。
  • 让字典在扩展字典种叫做 lib ，然后创建一个字典叫做视图中的字典。
  • 复制C:\Program Files\QuickTime\QTSystem\QTJavaNative.dll 和C:\Program Files\QuickTime\QTSystem\QTJNative.dll 到视图 字典。

你也需要安装一个数值转换器，我们可以提供 WinVDIG 在此下载：

http://www.eden.net.nz/7/20071008/

你必须要下载得是 1.0.1版本，而不是更新得版本。

然后你就可以通过在QTCap中的应用测试 QTJ 和数值转换器是不是安装正确。(如果你有困难，可以看 联系我们。)

在视图里面你要用管理员登陆然后打开NetLogo才可以使用视频得扩展。你也可以右击可执行和选择“用管理员身份登陆”。你也可以永久得改变这个设置：打开设置对话框，选择兼容建然后点“用管理员身份登陆”。

Mac OS X

为了让视频扩展可以使用，我们要将装入QuickTime得一些文件移动到扩展词典里。

  • 复制/System/Library/Java/Extensions/QTJava.zip 到扩展字典然后重命名为 QTJava.jar。
  • 在QTJ扩展字典中定一个字典叫 lib 然后叫 Mac OS X 中的 lib.
  • 复制/System/Library/Java/Extensions/libQTJNative.jnilib 到 Mac OS X 字典。

然而，在64位的 Macs (拥有当前所有得固体模型)，你会需要改变NetLogo来运行32位的模式来使用扩展。如果你不需要得话，通常得结果是会弹出 java.lang.UnsatisfiedLinkError 对话框。以下是操作步骤：

  • 在Finder上选择NetLogo 应用。
  • 在文件菜单上， 选择获取信息。
  • 选择复选框： "打开 32位模式" 。
  • 打开应用。

如果你需要，你先可以复制一份应用然后按照副本中所说得去做，但是开始还是需要打开64位的模式。（64位模式是推荐的默认,除非你正在使用该扩展。）

Linux

　QuickTime的扩展是不兼容的Linux,因为苹果没有一个Linux版本的QuickTime。

原语

qtj:camera-image qtj:camera-select qtj:camera-start qtj:camera-stop qtj:movie-close qtj:movie-image qtj:movie-open qtj:movie-open-player qtj:movie-set-time qtj:movie-start qtj:movie-stop

qtj:camera-image

qtj:camera-image

　返回一个图像从当前打开相机适合进口与位图扩展

qtj:camera-select

qtj:相机选择 width height

提供了一个对话框的安装摄像头,你可以从中选择。当您选择一个连接打开与选择的相机在给定的决议。如果你使用相机选择你不需要调用相机开始。你必须调用要么选择或之前就开始使用相机图像。

qtj:camera-start

qtj:camera-start width height

利用给定的方案打开链接的相机，然后做好准备去捕获图像。我们通常会在使用相机图片前，调用 相机开始 或者相机选择 。你不应该在相机已经打开得时候选择 相机开始 。在选择另一个相机前要先选择 相机停止 然后选择相机选择.

qtj:camera-stop

qtj:camera-stop

关闭所链接得相机。你应该在关闭NetLogo之前，或者是打开一个新得模型之前关闭相机。

qtj:movie-close

qtj:movie-close

关闭当前打开的电影，释放任何和当前播放窗口有关得资源，你应该在打开一个新的电影或者是关闭NetLogo之前关闭当前的电影。

qtj:movie-image

qtj:movie-image

返回一个电影中满足位图的扩展合适插入的当前窗体得图像。

qtj:movie-open

qtj:movie-open width height

载入具有特殊分辨率（在小方块中给出）的图像。

qtj:movie-player

qtj:movie-open-player

当你可以观看电影得时候开打一个视窗。

qtj:movie-set-time

qtj:movie-set-time seconds

根据给定的时间大小来改变电影的画面。

qtj:movie-start

qtj:movie-start

开始演电影。

qtj:movie-stop

qtj:movie-stop

暂停播放。
