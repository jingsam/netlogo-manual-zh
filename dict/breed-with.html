link-with

<breed>-with turtle
link-with turtle

Report the undirected link between turtle and the caller. If no link exists then it reports nobody. 

crt 2
ask turtle 0 [
  create-link-with turtle 1
  show link-with turtle 1 ;; prints link 0 1
]
