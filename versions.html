更新历史

NetLogo 5.0 用户手册

以下是一个缩略的历史。 详细的版本注释 在GitHub。

需要寻求旧版本上运行程序的帮助，查看 迁移指南。

Version 5.0 (February 2012)

  • 功能：
    • 开源(GPL license; 网上源程序在 https://github.com/NetLogo/NetLogo)
    • 模型实例和课程模型的新许可是创作共同署名非商业使用相同方式共享。
    • 支持应用国际字符（Unicode）
    • GUI有了西班牙语，俄语和汉语版本。（志愿翻译者所为）
    • 信息选项卡中丰富的格式和图像减少了
    • 画图程序在画图框中运行
    • 可以修改模型运行速度（调节帧速率的设置）
    • 按钮在开始计时前都可以选择不能用
    • 3D视图和Netlogo3D中可以使用半透明的颜色（只能用RGB颜色）
  • 语言改变
    • “任务”存储稍后运行的代码
      • 又叫做第一类函数，终结，lambda
      • 新的原语： task, is-command-task?, is-reporter-task?
      • 这些原语接受任务： run, runresult, foreach, map, reduce, filter, n-values, sort-by
    • 总体上提高了列表的性能（很多操作不再使用线性的时间，而是使用接近恒定的时间）
    • 在使用tick或tick-advance之前，必须使用reset-ticks开始计时
    • 增加了新的原语setup-plots和update-plots
    • 新的原语sort-on可以这样使用，例如：sort-on [size] turtles
    • 新的原语error造成了运行时间错误
    • random-normal避免了负的标准差
  • HubNet:
    • activities can run headless
    • 增加了新的原语hubnet-clients-list, hubnet-kick-client, hubnet-kick-all-clients
    • 不再需要hubnet-set-client-interface
  • 其他修订和改变：
    • tabs改名为 Interface/Info/Code
    • 命令中心允许返回自动加入 show 命令
    • NetLogo 3D 使用 .nlogo3d 而不是 .nlogo
    • import-world更快了
    • startup不再无目的运行或者在背景中，而在行为空间中运行
    • 修订的 3D 中小乌龟颜色过浅的错误
    • Pack200 允许在java小程序中快速装载
    • Mac app 默认以64 bit 模式运行
    • 升级了的 Windows 安装
    • GoGo 扩展开始经验在所有平台上更加平稳
